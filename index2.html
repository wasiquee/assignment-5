Question-1: What are the different ways to select an element in the DOM?


Selecting by Id (getElementById),Class(getElementsByClassName),Tag Name (getElementsByTagName),CSS Selectors (querySelector),Multiple Elements (querySelectorAll) and so many.

Question-2: What is the difference between innerHTML, innerText, and textContent ?


1- innerHTML=== Reads or writes the entire HTML inside an element.
2- innerText=== Contains only the visible text inside an element.
3- textContent=== Contains all text content, including hidden text.

Question-3: What is event delegation in the DOM?


It's an awesome technique in JavaScript where we can add a single event listener to a parent element instead of adding multiple child elements.
So that the number of event listeners will reduce

Question-4: What is event bubbling in the DOM?


Event Bubbling হলো যদি একটি বাটনে ক্লিক করা হয় তাহলে Event টি বাটনের উপর ঘটবে এবং সেটি Parent element (যেমন একটি div)-এ বাবল হয়ে যাবে। তারপর এটি উপরের দিকে যায়, যতক্ষণ না এটি পেইজের টপ-এ পোঁছায় (যেমন  document)
